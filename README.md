# CatDiary





## How to Build?



Setup:

1. Git clone the repo
2. `python -m venv env`
3. `source env/bin/activate`
4. `pip install -r requirements.txt`
5. `cd frontend` and `npm install`
6. 

Start:

Run the Flask backend:

```
cd backend
python server.py
```



Run the React development server:

```
cd frontend
npm run dev
```
